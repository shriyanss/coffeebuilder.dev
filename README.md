**Build Your Perfect Cup.**

Coffee Builder is a modern, interactive web application designed to help coffee lovers discover, customize, and brew their ideal coffee. Whether you're a home barista or just looking for something new, Coffee Builder helps you find the perfect recipe based on your preferences and available equipment.

## ‚ú® Features

- **Smart Recipe Discovery**: Filter recipes by milk preference, chocolate craving, temperature (hot/cold), and cup type.
- **Equipment Matching**: Input the equipment you own (Espresso Machine, French Press, V60, etc.) to only see recipes you can actually make.
- **Grind Size Guide**: Interactive slider to visualize and understand grind sizes for different brewing methods (from Turkish fine to Cold Brew coarse).
- **Detailed Instructions**: Step-by-step guides with precise proportions for coffee, water, milk, and foam.
- **Dark Mode**: Fully supported dark/light theme toggling.
- **Mobile First**: Optimized for use on your phone while brewing in the kitchen.

## üöÄ Getting Started

First, clone the repository:

```bash
git clone https://github.com/shriyanss/coffeebuilder.dev.git
cd coffeebuilder.dev
```

Install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üêõ Issues & Contributions

Found a bug in the code? Or maybe you spotted a mistake in a recipe (like a wrong grind size or proportion)? We'd love your help to fix it!

### How to Report an Issue

1. Go to the [Issues](https://github.com/shriyanss/coffeebuilder.dev/issues) tab on GitHub.
2. Click on **New Issue**.
3. Choose the appropriate template (or just start a blank one).
4. **For Code Issues**: Describe the bug, steps to reproduce, and expected behavior.
5. **For Recipe Issues**: Please specify the Recipe Name and what needs to be changed (e.g., "Flat White should have less foam").

We welcome contributions! Feel free to open a Pull Request if you want to add a new recipe or improve the app.
